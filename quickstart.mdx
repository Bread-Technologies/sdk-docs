Get the Bread SDK installed and authenticated so you can start baking AI models.

<Note>
New to baking? Start with [Understanding Baking](/understanding-baking) to learn the core concepts.
</Note>

## Install the SDK

Install the Bread Python SDK via pip:

<CodeGroup>

```bash MacOS/Linux
pip install aibread
```

```cmd Windows (Command Prompt)
pip install aibread
```

```powershell Windows (PowerShell)
python -m pip install aibread
```

</CodeGroup>


## Set Your API Key

After obtaining your API key, set it as an environment variable in your terminal: 

<CodeGroup>

```bash MacOS/Linux
export BREAD_API_KEY='your_api_key_here'
```

```cmd Windows (Command Prompt)
setx BREAD_API_KEY "your_api_key_here"
```

```powershell Windows (PowerShell)
$env:BREAD_API_KEY='your_api_key_here'
```

</CodeGroup>

## Initialize the Client

Create a Bread client that reads from your environment variable:

```python
import os
from aibread import Bread

client = Bread() # Automatically uses BREAD_API_KEY
```

<Tip>
For production deployments and `.env` file usage, see [Production Authentication](/authentication).
</Tip>

## Verify Your Setup

Test your connection with a health check and list your repos (new users should be 0):

```python
# Test connection
health = client.health.check()
print(f"Status: {health.status}")
print(f"API Version: {health.version}")

# List number of repos
response = client.repo.list()
print(f"Connected! Found {len(response.repos)} repositories")
```
