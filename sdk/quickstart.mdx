Use the **Bread SDK** for programmatic control over the baking process. Available in **Python** and **TypeScript**. Perfect for automation, integration, and advanced use cases.

<Note>
**Prefer a git-based workflow?** Check out [bgit Quickstart](/bgit/quickstart) for a git-native interface.
</Note>

## Installation

<CodeGroup>

```bash Python
# Install from PyPI
pip install aibread

# Or using uv
uv pip install aibread

# Or install from source
git clone https://github.com/Bread-Technologies/Bread-SDK-Bake-Repo.git
cd Bread-SDK-Bake-Repo
pip install -e .
```

```bash TypeScript
# Install from npm
npm install @aibread/sdk

# Or using yarn
yarn add @aibread/sdk
```

</CodeGroup>

## Authentication

Set your API key as an environment variable:

```bash
export BREAD_API_KEY=your-api-key-here
```

Or use a `.env` file (see [Authentication](/authentication) for details).

## Your First Bake

<CodeGroup>

```python Python
import os
from aibread import Bread

# Initialize client
client = Bread(api_key=os.environ.get("BREAD_API_KEY"))

# Set repository
client.repo.set(repo_name="my_repo", base_model="Qwen/Qwen3-32B")

# Set prompt
client.prompts.set(
    prompt_name="yoda_teacher",
    repo_name="my_repo",
    messages=[
        {"role": "system", "content": "You are Yoda. Speak like Yoda."}
    ]
)

# Set target
client.targets.set(
    target_name="main_target",
    repo_name="my_repo",
    template="default",
    overrides={
        "generators": [
            {"type": "oneshot_qs", "model": "claude-sonnet-4-5-20250929", "numq": 100}
        ],
        "model_name": "Qwen/Qwen3-32B",
        "teacher_prompt": "yoda_teacher",
        "student_prompt": ""
    }
)

# Run stim
client.targets.stim.run(
    target_name="main_target",
    repo_name="my_repo"
)

# Run rollout
client.targets.rollout.run(
    target_name="main_target",
    repo_name="my_repo"
)

# Set bake
client.bakes.set(
    bake_name="v1",
    repo_name="my_repo",
    template="default",
    overrides={
        "datasets": [{"target": "main_target", "weight": 1.0}]
    }
)

# Run bake
client.bakes.run(bake_name="v1", repo_name="my_repo")
```

```typescript TypeScript
import Bread from '@aibread/sdk';

// Initialize client
const client = new Bread({
  apiKey: process.env.BREAD_API_KEY,
});

// Create repository
await client.repo.create({
  repo_name: 'my_repo',
  base_model: 'Qwen/Qwen3-32B',
});

// Create prompt
await client.prompts.create('my_repo', {
  prompt_name: 'yoda_teacher',
  messages: [
    { role: 'system', content: 'You are Yoda. Speak like Yoda.' }
  ],
});

// Create target
await client.targets.create('my_repo', {
  target_name: 'main_target',
  template: 'default',
  overrides: {
    generators: [
      { type: 'oneshot_qs', model: 'claude-sonnet-4-5-20250929', numq: 100 }
    ],
    model_name: 'Qwen/Qwen3-32B',
    teacher_prompt: 'yoda_teacher',
    student_prompt: '',
  },
});

// Run stim
await client.targets.stim.run('my_repo', {
  target_name: 'main_target',
});

// Run rollout
await client.targets.rollout.run('my_repo', {
  target_name: 'main_target',
});

// Create bake
await client.bakes.create('my_repo', {
  bake_name: 'v1',
  template: 'default',
  overrides: {
    datasets: [{ target: 'main_target', weight: 1.0 }],
  },
});

// Run bake
await client.bakes.run('v1', {
  repo_name: 'my_repo',
});
```
